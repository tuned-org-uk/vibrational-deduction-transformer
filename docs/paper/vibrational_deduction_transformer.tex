\begin{abstract}
We present the \emph{Wiring Autoencoder v2} (WAE\,v2), a fully Bayesian
generative model that upgrades the original WAE family with three coupled
spectral priors derived from probabilistic PCA in a Laplacian eigenbasis
(Spectral-PPCA).  The v1 architecture used a hard high-frequency energy
penalty ($J_{\mathrm{freq}}$) and an isotropic Gaussian latent prior.  WAE\,v2
replaces both with principled variational KL terms: (i) a spectral-basis KL
over the loading matrix $\bS$ weighted by Laplacian eigenvalues, implementing a
\emph{spectral Occam's razor}; (ii) a closed-form Gamma/Exponential KL over
per-mode weights $\bomega$, providing soft $\tau$-mode band-limiting; and (iii)
the standard isotropic Gaussian latent KL retained for stability (the
Laplacian-precision latent KL is preserved as a design option and ablation, but
is not included in the default three-term ELBO following architectural review in
PR\,\#35).  After training, a \emph{spectral artefact} $\A(\I)$ is extracted
and used to initialise a transformer with pre-built spectral associative memory.
The two-phase (offline training, online inference) architecture separates
spectral learning from deployment, with frozen eigenpairs eliminating all
runtime $O(N^3)$ eigendecompositions during training.  We detail the complete
module design, three-term ELBO, stability analysis, benchmark metrics, and
associative memory construction.
\end{abstract}

\section{Introduction}
\label{sec:intro}

Graph-structured latent variable models occupy an increasingly important niche
between classical variational autoencoders (VAEs) and geometric deep learning:
they can encode both the \emph{content} of nodes and the \emph{relational
structure} that organises them.  The original Wiring Autoencoder~\citep{wae_v1}
recognised this, using a differentiable graph Laplacian $L(z)$ as an
intermediate latent object whose spectral properties are regularised by a hard
frequency penalty $J_{\mathrm{freq}} = \sum_{j>k}\lambda_j(L(z))$.  While
effective in practice, this penalty is a heuristic rather than a proper
probabilistic prior, which makes it difficult to:
\begin{enumerate}
  \item compare competing ArrowSpace indices $\I_1, \I_2$ via principled model
        selection;
  \item interpret the learned spectral structure as a posterior over loading
        matrices $\bW$;
  \item extract a reusable \emph{spectral artefact} whose keys are
        approximately orthonormal and whose values are grounded in the data
        manifold.
\end{enumerate}

WAE\,v2 addresses all three limitations by embedding the Wiring Autoencoder
within the Spectral-PPCA framework~\citep{design_a}, a Bayesian generative
model in which the loading matrix $\bW = \bU_{1:q}\,\diag(\bomega)\,\bS$
lives explicitly in the Laplacian eigenbasis and is endowed with
eigenvalue-weighted shrinkage priors.

\subsection{Contributions}
\begin{itemize}
  \item A three-term ELBO replacing $J_{\mathrm{freq}}$ and preserving the
        isotropic latent prior (Section~\ref{sec:elbo}).
  \item A \texttt{SpectralLoadingDecoder} that maps latent codes to
        eigenbasis-factorised loading matrices (Section~\ref{sec:modules}).
  \item A \texttt{WiringEncoderV2} with VDT recurrence and
        \texttt{ModeWeightHead} producing variational Gamma parameters
        (Section~\ref{sec:modules}).
  \item A \texttt{SpectralAssociativeMemory} module that pre-builds a Hopfield
        memory from the spectral artefact (Section~\ref{sec:memory}).
  \item A two-phase offline/online architecture eliminating runtime
        eigensolves during training (Section~\ref{sec:two_phase}).
  \item An eight-metric evaluation suite including ELBO Bayes factors for
        ArrowSpace index selection (Section~\ref{sec:benchmarks}).
\end{itemize}

\section{Background}
\label{sec:background}

\subsection{Probabilistic PCA}

Standard PPCA~\citep{tipping1999ppca} defines a linear-Gaussian generative model:
\begin{equation}
  \bz \sim \N(\mathbf{0}, \bm{I}_q), \qquad
  \bx \mid \bz \sim \N(\bW\bz + \bmu,\, \sigma^2 \bm{I}_d),
  \label{eq:ppca}
\end{equation}
producing marginal $\bx\sim\N(\bmu,\,\bW\bW^\top+\sigma^2\bm{I}_d)$.
Maximum-likelihood estimation recovers the PCA loading matrix in the column
space of the $q$ largest singular vectors of the data matrix.  The fully
Bayesian extension places priors over $\bW$, $\bz$, and $\sigma^2$, enabling
model selection over the latent dimensionality $q$ via the marginal
likelihood~\citep{minka2001pca}.

\subsection{Graph Laplacian Spectral Theory}

Let $G=(V,E,w)$ be a weighted graph with node set $|V|=N$.  The normalised
graph Laplacian is $L = D^{-1/2}(D-W)D^{-1/2}$ where $D_{ii}=\sum_j w_{ij}$.
Eigendecomposition $L = \bU\bLambda\bU^\top$ yields eigenvectors
$\bU=(\mathbf{u}_1,\dots,\mathbf{u}_N)\in\R^{N\times N}$ and non-negative
eigenvalues $0\le\lambda_1\le\cdots\le\lambda_N$.  Low-frequency eigenvectors
(small $\lambda_k$) are smooth over $G$; high-frequency ones (large $\lambda_k$)
oscillate rapidly across edges~\citep{chung1997spectral}.

Given ArrowSpace index $\I$, we denote the frozen $q$-truncated eigendecomposition
\begin{equation}
  L(\I) = \bU\bLambda\bU^\top,
  \qquad
  \bU_q = \bU_{:,1:q}\in\R^{N\times q},
  \qquad
  \bLambda_q = \diag(\lambda_1,\dots,\lambda_q).
  \label{eq:eig}
\end{equation}
These eigenpairs are computed once offline and used as frozen constants
throughout training.

\subsection{Wiring Autoencoder (WAE)}
\label{sec:wae_v1}

WAE\,v1 trains a differentiable Laplacian $L(\bz)$ whose topology is fixed
(kNN + RBF graph) but whose edge weights are learned by a mixture-of-experts
decoder $W_{\mathrm{dec}}:\bz\mapsto\delta_e$.  The training objective is
\begin{equation}
  \elbo_{\mathrm{WAE}} =
    \E_q[\log p(\bx\mid\bz)] -
    \beta\cdot\KL(q(\bz)\|\N(\mathbf{0},\bm{I})) -
    \alpha\cdot J_{\mathrm{freq}}(L(\bz)),
  \label{eq:wae_v1}
\end{equation}
where $J_{\mathrm{freq}} = \sum_{j>k}\lambda_j(L(\bz))$ penalises
high-frequency wiring.  The reconstruction is provided by
\texttt{TauModeDiffusion}: the heat kernel
$K_\tau = \bU_k\exp(-t\bLambda_k)\bU_k^\top$ applied to an embedding table
$\bE$, with learnable diffusion time $t$.

Three limitations of this design motivate WAE\,v2:
(a) $J_{\mathrm{freq}}$ is a deterministic penalty with no prior interpretation;
(b) the latent prior is isotropic and ignores the graph geometry;
(c) there is no principled post-training artefact for downstream use.

\section{Spectral-PPCA Generative Framework}
\label{sec:spectral_ppca}

Following~\citet{design_a}, we define the Spectral-PPCA prior by replacing the
standard PPCA loading matrix with one that lives in the Laplacian eigenbasis.

\subsection{Three Structural Priors}

\paragraph{Prior 1: Spectral-basis loading.}
Parametrise $\bW = \bU_{1:q}\,\diag(\bomega)\,\bS$ where
$\bU_{1:q}\in\R^{d\times q}$ are the $q$ lowest-frequency eigenvectors of
$L(\I)$ (frozen) and $\bS\in\R^{q\times q}$ lives in the spectral basis.
Place an eigenvalue-weighted Gaussian shrinkage prior on $\bS$:
\begin{equation}
  p(\bS\mid\I) \propto
  \exp\!\left(-\tfrac{\lambda_s}{2}\tr(\bS^\top \bLambda_q \bS)\right).
  \label{eq:prior_S}
\end{equation}
This implements a \emph{spectral Occam's razor}: high-frequency components
(large $\lambda_k$) are penalised exponentially more strongly than low-frequency
ones, pulling the loading matrix toward smooth directions of the index graph.

\paragraph{Prior 2: $\tau$-mode frequency prior.}
Let $\omega_k > 0$ be a weight on the $k$-th mode.  The $\tau$-mode prior is a
product of exponentials with rates that scale with Laplacian eigenvalues:
\begin{equation}
  p(\omega_k\mid\tau,\lambda_k) = \mathrm{Exp}(\tau\lambda_k),
  \qquad k=1,\dots,q.
  \label{eq:prior_omega}
\end{equation}
Small $\lambda_k$ (low-frequency modes) receive weak penalisation; large
$\lambda_k$ receive strong penalisation, naturally selecting the effective
spectral bandwidth.  The loading matrix is a mode-weighted linear combination
$\bW = \bU_{1:q}\diag(\bomega)\bS$.

\paragraph{Prior 3 (design option): Laplacian-precision latent prior.}
For a richer latent geometry one can replace the isotropic prior with a
Laplacian-precision prior:
\begin{equation}
  p(\bz\mid\I) = \N(\mathbf{0}, (\bm{I}+\beta L_s)^{-1}),
  \label{eq:prior_z_lap}
\end{equation}
where $L_s$ is a sample-graph Laplacian built from the current batch of latent
means (stop-gradient).  This enforces smoothness of latent codes over the
data manifold.  The resulting KL expands to
\begin{equation}
  \KL(q(\bz)\|p_{\mathrm{Lap}}(\bz)) = \tfrac{1}{2}\!
  \left[
    \tr(\Sigma_z(\bm{I}{+}\beta L_s))
    + \bmu_z^\top(\bm{I}{+}\beta L_s)\bmu_z
    - d
    - \log\det\Sigma_z
    + \log\det(\bm{I}{+}\beta L_s)^{-1}
  \right],
  \label{eq:kl_lap}
\end{equation}
where the cross-term $\bmu_z^\top L_s\bmu_z$ is the graph Dirichlet energy of
the latent means.  \emph{This prior is preserved in the WAE\,v2 codebase as an
ablation option} (\texttt{use\_isotropic\_kl=False} in
\texttt{WiringEncoderV2}) but is \textbf{not} included in the default
three-term ELBO following architectural review in PR\,\#35.

\section{WAE v2: Three-Term ELBO}
\label{sec:elbo}

\subsection{Variational Family}

We adopt a mean-field amortised variational family:
\begin{align}
  q_\phi(\bz\mid\bx) &= \N(\bmu_z, \diag(\exp(\bm{s}_z))), \\
  q_\phi(\bS\mid\bx) &= \N(M_S, \diag(\exp(\bm{v}_S))), \\
  q_\phi(\omega_k\mid\bx) &= \mathrm{Gamma}(a_k, b_k),
\end{align}
where all parameters are outputs of the encoder network \texttt{WiringEncoderV2}.
The Gamma distribution is reparametrisable via the implicit reparametrisation
trick~\citep{figurnov2018implicit}; in practice we parametrise through
$(\log a_k, \log b_k)$ from a \texttt{ModeWeightHead} linear layer.

\subsection{ELBO Decomposition}

The default WAE\,v2 training objective is the following three-term ELBO
(minimisation form, as implemented in \texttt{wae/model.py}):
\begin{equation}
  \boxed{
  \mathcal{L}_{\mathrm{WAE-v2}}
  = \underbrace{\E_q[\|\bx - \hat{\bx}\|^2 / 2\sigma^2]}_
      {\text{reconstruction}}
  + \underbrace{\KL\bigl(q(\bz)\|\N(\mathbf{0},\bm{I})\bigr)}_
      {\text{isotropic latent KL }(\mathtt{kl\_z})}
  + \underbrace{\KL\bigl(q(\bS)\|p(\bS\mid\I)\bigr)}_
      {\text{spectral basis KL }(\mathtt{kl\_S})}
  + \underbrace{\KL\bigl(q(\bomega)\|p(\bomega\mid\tau,\bLambda)\bigr)}_
      {\tau\text{-mode KL }(\mathtt{kl\_tau})}
  }
  \label{eq:elbo_v2}
\end{equation}
All four terms are non-negative; the total loss is minimised (ELBO is
maximised by negating) during gradient descent.

\subsection{Closed-Form KL Terms}

\paragraph{Isotropic latent KL.}
\begin{equation}
  \KL(q(\bz)\|\N(\mathbf{0},\bm{I})) =
  -\tfrac{1}{2}\sum_{i=1}^{d_z}\!
  \bigl(1 + s_{z,i} - \mu_{z,i}^2 - e^{s_{z,i}}\bigr).
  \label{eq:kl_z}
\end{equation}

\paragraph{Spectral-basis KL.}
For the entry $(k,j)$ of $\bS$, with precision $\lambda_s\lambda_k$
(\cref{eq:prior_S}):
\begin{equation}
  \KL_{kj} = \tfrac{1}{2}\bigl[
    \lambda_s\lambda_k(v_{S,kj} + S_{kj}^2)
    - v_{S,kj} - 1 - \log(\lambda_s\lambda_k)
  \bigr],
  \label{eq:kl_S}
\end{equation}
where $v_{S,kj} = \exp(\log\mathrm{var}_{S,kj})$.  In the current
implementation, the log-variance is approximated as
$\log\mathrm{var}_{S,kj} = \log(S_{kj}^2 + \epsilon)$, deriving uncertainty
directly from the magnitude of the mean --- a simplification that avoids adding
a separate posterior variance head to \texttt{SpectralLoadingDecoder}.  Future
work should add a dedicated variance head for a proper amortised posterior.

\paragraph{$\tau$-mode KL.}
For mode $k$ with $q(\omega_k)=\mathrm{Gamma}(a_k, b_k)$ and
$p(\omega_k)=\mathrm{Gamma}(1, \tau\lambda_k)$, the closed-form KL is:
\begin{equation}
  \KL(\mathrm{Gamma}(a,b)\|\mathrm{Exp}(r)) =
  \log b - \log r + \ln\Gamma(a) + (1-a)\psi(a) + \frac{ab}{r},
  \quad r = \tau\lambda_k,
  \label{eq:kl_tau}
\end{equation}
where $\psi$ is the digamma function.  This is summed over all $q$ modes and
averaged over the batch.

\subsection{WiringEncoderV2}
\label{sec:encoder}

\texttt{WiringEncoderV2} (\texttt{wae/encoder.py}) replaces the v1 MLP encoder
with a VDT (Vibrational Discrete Transformer) recurrence stack:
\begin{enumerate}
  \item \textbf{Input projection}: linear layer mapping $(B,D)\to(B,N{\cdot}d)$,
        reshaped to $(B,N,d)$.
  \item \textbf{VDT blocks}: $K$ \texttt{VibrationalStateBlock} layers operating
        on $(B,N,d)$ state matrices using the feature-space Laplacian $L_f$.
        The recurrence implements a discrete damped wave equation in the graph
        spectral domain.
  \item \textbf{Modal projection}: mean-pool output over the node dimension
        after projecting through eigenvectors $\bm{v}\in\R^d$.
  \item \textbf{Readout heads}: separate linear layers producing
        $\bmu,\log\bm{s}$ (for $q(\bz)$) and \texttt{ModeWeightHead}
        producing $(\log\bm{a},\log\bm{b})$ for $q(\bomega)$.
\end{enumerate}
The optional $\lambda$-fingerprint (ArrowSpace spectral histogram of $L(\I)$)
can be concatenated to $\bx$ before the input projection
(\texttt{use\_lambda\_features=True}).

\paragraph{ModeWeightHead.}
A single linear layer \texttt{Linear(feat\_dim, 2q)} splits into
$(\log\bm{a}, \log\bm{b})\in\R^q\times\R^q$.  The Gamma parameters are
obtained as $a_k = e^{\log a_k}>0$, $b_k = e^{\log b_k}>0$.
The mode-weight KL \eqref{eq:kl_tau} is computed from these in
\texttt{wae/spectral.py:tau\_mode\_kl}.

\subsection{SpectralLoadingDecoder}
\label{sec:decoder}

\texttt{SpectralLoadingDecoder} (\texttt{wae/wiring\_decoder.py}) replaces
\texttt{WiringDecoder} as the default v2 wiring decoder.  The forward pass
computes:
\begin{align}
  \bS &= \texttt{S\_net}(\bz).\text{view}(B,q,q), \label{eq:S_net}\\
  \bomega &= \exp(\texttt{omega\_net}(\bz)), \label{eq:omega_net}\\
  \bW &= \bU_q^\top \otimes (\bomega_{k}\bS),\quad
        \text{i.e.}\quad
        \bW = \bU_q.\text{unsqueeze}(0)
        \;\texttt{@}\;(\bomega.\text{unsqueeze}(-1)\ast\bS),
        \label{eq:W}\\
  L(\bz) &= \texttt{DifferentiableLaplacian.from\_spectral\_loading}(\bW, L_{\mathrm{base}}).
           \label{eq:Lz}
\end{align}

Edge weights in \eqref{eq:Lz} are synthesised as
$w_{ij} = w_{ij}^{\mathrm{base}}\cdot\sigma(-\|\bW_i-\bW_j\|^2)$ where
$\bW_i$ is the $i$-th row of $\bW$, and $\sigma$ is the logistic sigmoid.
This path is fully differentiable from $L(\bz)$ back to $\bz$ through $\bW$.

\paragraph{Initialisation.}
\texttt{S\_net.weight} is initialised so the initial $\bS_b\approx\bm{I}_q$
(near-identity); \texttt{omega\_net} is initialised near zero so initial
$\omega_k\approx 1$ for all modes.  This gives a warm-start close to the
eigenbasis reparametrisation $\bW\approx\bU_q$.

\paragraph{Config dispatch.}
The v1 \texttt{WiringDecoder} (mixture-of-experts) is retained as a
fallback: \texttt{decoder\_type: spectral} selects
\texttt{SpectralLoadingDecoder}; \texttt{decoder\_type: mixture\_of\_experts}
selects \texttt{WiringDecoder}.

\subsection{DiffusionDecoder (Unchanged)}

\texttt{DiffusionDecoder} is identical to v1.  It wraps
\texttt{TauModeDiffusion}: the truncated heat kernel
\begin{equation}
  K_\tau = \bU_k\,\diag(e^{-t\bLambda_k})\,\bU_k^\top,
  \label{eq:heat_kernel}
\end{equation}
with learnable log-diffusion-time $\log t$.  A small MLP refinement
residually corrects the output.  Reconstruction loss is Gaussian NLL:
$\|\bx-\hat{\bx}\|^2/(2\sigma^2)$.

\subsection{WiringAutoencoderV2}

\texttt{WiringAutoencoderV2} (\texttt{wae/model.py}) assembles all three
modules.  Its \texttt{forward()} method returns a dict with exactly nine keys:
\texttt{\{loss, recon, kl\_z, kl\_S, kl\_tau, x\_hat, z, mu, log\_var\}}.
The \texttt{L\_z} key present in v1 is not returned (PR\,\#35: no runtime
Laplacian is exposed).

\paragraph{Model factory.}
\texttt{from\_config(cfg, E)} dispatches on \texttt{cfg[`model'][`version']}:
version~1 builds \texttt{WiringAutoencoder}; version~2 builds
\texttt{WiringAutoencoderV2}.

\section{Spectral Artefact and Associative Memory}
\label{sec:memory}

\subsection{Spectral Artefact}

After training to convergence, the spectral artefact $\A(\I)$ is extracted
by evaluating the model at the prior mean $\bz=\mathbf{0}$:
\begin{equation}
  \A(\I) = \left\{
    \hat{\bW} = \bU_q\,\diag(\hat{\bomega})\,\hat{\bS},
    \quad
    \hat{\bomega} = (\hat{\omega}_1,\dots,\hat{\omega}_q),
    \quad
    S_{\I} = \sum_{k=1}^q \hat{\omega}_k\,\hat{\bm{w}}_k\hat{\bm{w}}_k^\top
  \right\},
  \label{eq:artefact}
\end{equation}
where $\hat{\bm{w}}_k$ is the $k$-th column of $\hat{\bW}$.
Note that \eqref{eq:artefact} uses $d_{\theta}(\hat{\bm{w}}_k)\approx\hat{\bm{w}}_k$
(linear decoder approximation for $S_{\I}$); a full nonlinear value function
$d_{\theta}(\hat{\bm{w}}_k)$ is left as future work.

\label{tab:artefact}

\subsection{SpectralAssociativeMemory}
\label{sec:assoc_mem}

\texttt{SpectralAssociativeMemory} (\texttt{wae/spectral\_memory.py}) wraps
$S_{\I}$ as a pre-built Hopfield / linear associative memory.  For query
$\bm{q}\in\R^{d_k}$, retrieval proceeds via:
\begin{equation}
  \bm{o} = \mathrm{softmax}\!\left(\beta_H S_{\I}^\top \bm{q}\right) S_{\I},
  \label{eq:hopfield}
\end{equation}
where $\beta_H$ is the inverse temperature controlling retrieval sharpness.
The approximately-orthonormal keys $\hat{\bm{w}}_k$ (columns of $\hat{\bW}$
are linear combinations of eigenvectors, orthonormal when $\bS\approx\bm{I}$)
maximise retrieval SNR~\citep{ramsauer2020hopfield}:
$\mathrm{SNR}^{-1}\approx N_{\mathrm{stored}}/d_k$.

\paragraph{Online delta-rule updates.}
New associations can be written without overwriting spectral structure:
\begin{equation}
  S_{\I} \leftarrow S_{\I} + \tfrac{1}{B}(\bm{v}-S_{\I}\bm{k})\,\bm{k}^\top,
  \label{eq:delta_rule}
\end{equation}
where $(\bm{k},\bm{v})$ are new key-value pairs.

\section{Two-Phase Architecture}
\label{sec:two_phase}

WAE\,v2 cleanly separates spectral learning from online inference.

\begin{algorithm}[h]
\caption{WAE\,v2 Two-Phase Procedure}
\label{alg:two_phase}
\begin{algorithmic}[1]
\Require ArrowSpace index $\I$, dataset $D$, hyperparameters
\State \textbf{// Phase 1 --- Offline}
\State Compute eigendecomposition: $(\bU_q,\bLambda_q)\leftarrow\texttt{eigh}(L(\I))$ \Comment{once; $O(N^3)$}
\State Freeze $(\bU_q,\bLambda_q)$
\For{each training step}
  \State $\bz,\bmu,\log\bm{s},\log\bm{a},\log\bm{b}\leftarrow\texttt{WiringEncoderV2}(\bx)$
  \State $\bW,\bomega,\bS,L(\bz)\leftarrow\texttt{SpectralLoadingDecoder}(\bz,\bU_q,L_{\mathrm{base}})$
  \State $\hat{\bx}\leftarrow\texttt{DiffusionDecoder}(L(\bz),\bE)$
  \State Compute $\mathcal{L}_{\mathrm{WAE-v2}}$ (\cref{eq:elbo_v2}) and update $\phi,\theta$
\EndFor
\State $\A(\I)\leftarrow\texttt{extract\_spectral\_artefact}(\bU_q,\bLambda_q)$
\State $S_{\I}\leftarrow\texttt{SpectralAssociativeMemory}(\A(\I))$
\State \textbf{// Phase 2 --- Online}
\State Initialise transformer FFN / cross-attention from $S_{\I}$
\For{each inference query $\bm{q}$}
  \State Retrieve via \cref{eq:hopfield}; optionally update via \cref{eq:delta_rule}
\EndFor
\end{algorithmic}
\end{algorithm}

\paragraph{No runtime eigensolves.}
Because $(\bU_q,\bLambda_q)$ are frozen constants in Phase~1, \emph{no
Laplacian is computed or inverted at each training step}.  All $O(N^3)$
eigendecomposition cost is paid once in offline preprocessing.

\paragraph{ELBO Bayes factor for index selection.}
Training WAE\,v2 on competing indices $\I_1, \I_2$ yields ELBO estimates
$\elbo(\I_1),\elbo(\I_2)$.  The Bayes factor
\begin{equation}
  \mathrm{BF}_{12} = \exp(\elbo(\I_1) - \elbo(\I_2))
  \label{eq:bayes_factor}
\end{equation}
provides a principled comparison of index constructions: an index whose
spectral geometry explains the data covariance better will achieve a higher
ELBO~\citep{lotfi2022marginal}.

\section{Stability Analysis}
\label{sec:stability}

WAE\,v2 inherits the full v1 stability hierarchy from
\texttt{docs/v2/04-stability.md}.  Two additional conditions apply:

\paragraph{Sample-graph Laplacian feedback.}
When the Laplacian-precision latent KL (design option) is enabled, $L_s$ is
built from the batch of latent means $\bmu_z$, creating a feedback loop.
Prevention: apply stop-gradient on $L_s$ construction; or use a frozen
sample Laplacian refreshed every $T_{\mathrm{refresh}}$ epochs.

\paragraph{Spectral-basis KL conditioning.}
If $\lambda_1\approx 0$ (Fiedler vector near zero), the prior on the first
mode becomes nearly flat, increasing gradient variance.  Apply a floor
$\lambda_k\leftarrow\max(\lambda_k,\epsilon)$ with $\epsilon=10^{-3}$ in all
KL computations (already implemented in \texttt{spectral.py} as
\texttt{eigvals.clamp(min=1e-6)}).

\paragraph{Mode weight collapse.}
High-frequency modes may collapse to $\omega_k\to 0$ if $\tau\lambda_k$ is
large.  Monitor active mode count $|\{k : \E[\omega_k]>\delta\}|$; enforce a
soft floor on the Gamma shape $a_k\ge a_{\min}$.

\section{Algorithm Tracks}
\label{sec:tracks}

The WAE\,v2 framework supports six implementation tracks.  The recommended
sequence is 6~$\to$ 1~$\to$ 4~$\to$ 3.

\begin{itemize}
  \item \textbf{Option 1 --- Deterministic AE.}  Replace $J_{\mathrm{freq}}$
        with the soft $\tau$-mode KL; add \texttt{SpectralLoadingDecoder} as
        ablation.  Backward-compatible: $\beta\to 0$ recovers the deterministic
        limit.
  \item \textbf{Option 3 --- Latent diffusion.}  WAE\,v2 provides the Stage~1
        encoder; the spectral noise schedule
        $a_\tau^{(k)}=\exp(-\tau\lambda_k t)$ matches the $\bLambda_q^{-1}$
        latent prior.
  \item \textbf{Option 4 --- Variational Laplace AE.}  WAE\,v2 is the full
        MC-ELBO version; Laplace covariance
        $(H_{\mathrm{recon}}+\bLambda_m)^{-1}$ is the deterministic
        approximation to the spectral-precision posterior.
  \item \textbf{Option 6 --- Classifier/Reasoner.}  The signed density matrix
        $\varrho_t=\varrho_t^+-\varrho_t^-$ from the VDT backbone can be used
        to initialise $S_{\I}$, grounding associative memory in
        reasoning-time spectral structure.
\end{itemize}

\section{Relationship to Prior Work}
\label{sec:related}

WAE\,v2 integrates ideas from several lines of work:

\paragraph{Graph VAEs.}  \citet{kipf2016vgae} encode graph structure
into a VAE but do not impose Laplacian spectral priors on latent codes or
loading matrices.  WAE\,v2 makes the spectral structure explicit and
variational.

\paragraph{Spectral graph convolution.}  Graph spectral methods~\citep{defferrard2016chebynet}
use Laplacian eigenvectors as filter bases; WAE\,v2 places this basis \emph{in
the generative model} as the loading matrix parametrisation.

\paragraph{Laplacian priors.}  \citet{rao2008laplacian} and
\citet{slawski2015lapprior} use Laplacian quadratic-form priors for Bayesian
variable selection; WAE\,v2 applies this to both the loading matrix $\bS$
and (optionally) the latent code $\bz$.

\paragraph{Hopfield / associative memory.}  \citet{ramsauer2020hopfield}
extended Hopfield networks to continuous attractors; \citet{geva2021ffn} showed
that transformer FFN layers are key-value memories.  WAE\,v2 unifies these by
seeding the memory with spectral artefacts from a generative model.

\paragraph{VDT backbone.}  The encoder uses the Vibrational Discrete Transformer
from~\citet{moriondo2026vdt}, which implements a discrete damped wave equation
in the graph spectral domain as the recurrent state update.

\section{Discussion and Open Problems}
\label{sec:discussion}

\paragraph{Posterior variance head for $\bS$.}
The current implementation approximates $\log\mathrm{var}_{S,kj}\approx\log(S_{kj}^2+\epsilon)$.
Adding a dedicated variance head in \texttt{SpectralLoadingDecoder} would give
a proper amortised Gaussian posterior for $\bS$ and is a prioritised open
task.

\paragraph{Laplacian-precision latent KL.}
PR\,\#35 removed this term from the default ELBO after finding that the
feedback loop between $\bmu_z$ and the dynamically-constructed $L_s$
required careful stop-gradient discipline and added training instability.
A frozen sample Laplacian (refreshed every $T$ epochs) may resolve this;
experiments are ongoing.

\paragraph{Nonlinear value function.}
The current spectral artefact uses a linear decoder approximation
$d_\theta(\hat{\bm{w}}_k)\approx\hat{\bm{w}}_k$ for constructing $S_{\I}$.
Using the full \texttt{DiffusionDecoder} response at each spectral direction
would produce a proper key-value Hopfield structure and is left as future work.

\paragraph{Iterative eigensolver.}
For large graphs ($N\gg 10^3$), truncated LOBPCG or ARPACK can replace
full \texttt{eigh} in the offline phase, reducing the preprocessing cost
from $O(N^3)$ to $O(N\cdot q)$.

\section{Conclusion}
\label{sec:conclusion}

WAE\,v2 upgrades the Wiring Autoencoder from a heuristically regularised
generative model to a principled Bayesian VAE in which the latent space and
loading matrix are explicitly shaped by Laplacian spectral geometry provided by
an ArrowSpace index.  The three-term ELBO (Eq.~\ref{eq:elbo_v2}) replaces the
hard $J_{\mathrm{freq}}$ penalty with interpretable variational KL terms; the
two-phase offline/online architecture eliminates runtime eigensolves; and the
spectral artefact $\A(\I)$ provides a principled initialiser for transformer
associative memory.  ELBO Bayes factors make ArrowSpace index selection a
first-class modelling decision.
